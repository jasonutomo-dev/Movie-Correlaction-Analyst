# Movie-Correlaction-Analyst
Analyzing a movie dataset to understand how variables like budget, votes, and runtime influence gross revenue. Includes correlation analysis, data cleaning, and visualizations using Python and Jupyter Notebook.

Project ini menganalisis dataset film untuk memahami faktor yang memengaruhi pendapatan kotor (gross revenue). Variabel utama yang dianalisis antara lain budget, votes, runtime, dan score. 

Dari analisis, diketahui bahwa budget dan votes memiliki korelasi positif kuat dengan gross revenue (~0.75 dan ~0.63). Variabel lain seperti tahun rilis dan durasi juga dianalisis untuk melihat pola tambahan. 

Dataset berisi kolom seperti name (judul), year, genre, budget, gross, votes, dan runtime. Analisis ini dapat membantu memahami tren industri film dan memprediksi film yang berpotensi sukses.
